# Pixel Sampler — Changelog

## v1.0 (2026)

- Initial release
- Fonts browser: all system fonts, each row rendered in its own font, specimen detail screen
- Colors browser: full 64-color Pebble palette with swatches, GColor names, hex codes, full-screen detail
- Platform info screen: device name, screen size, shape, color depth, health capability
- All 7 Pebble platforms supported (aplite, basalt, chalk, diorite, emery, flint, gabbro)
