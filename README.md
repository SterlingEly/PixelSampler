# Pixel Sampler

A developer reference app for Pebble. Browse all system fonts, the full 64-color palette, and hardware platform details — all on your wrist.

## Features

- **Fonts** — Scrollable list with each row rendered in its own font. Select any font for a full-screen specimen with cycleable sample strings.
- **Colors** — All 64 Pebble palette colors with color swatch, GColor name, and hex code. Select any color for a full-screen preview with contrast-adaptive text.
- **Platform** — Current device info: platform name, screen dimensions, shape, color depth, health capability.

## Platforms

All 7 Pebble platforms: aplite, basalt, chalk, diorite, emery, flint, gabbro.
